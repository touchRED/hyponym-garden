this is hyponym garden, a tool for navigating image culture. that is, it displays words with their associated imagery.

----

operator's manual

right arrow -- move "forward" a synonym

left arrow -- move "back" to the previous synonym

up arrow -- move "up" to a hypernym

down arrow -- move "down" to a hyponym

space bar -- cycle through images

----

click on the current word to change it to whatever you want
